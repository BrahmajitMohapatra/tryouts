Class Tryouts::Tryout::DrillContext

Attributes

clean [R] A block to executed one time before starting the drills
command [R] For drill type :cli, this is the name of the command to test. It should be a valid method available to a Rye::Box object. For drill type :api, this attribute is ignored.
dream_catcher [R] A Hash of Dream objects for this Tryout. The keys are drill names.
drills [R] An Array of Drill objects
dtype [R] A default value for Drill.dtype
failed [R] The number of dreams that did not come true (failed drills)
name [R] The name of this tryout
passed [R] The number of dreams that came true (successful drills)
setup [R] A block to executed one time before starting the drills
skipped [R] The number of skipped drills
